# ClearPact

[![API](https://img.shields.io/badge/API-v1.0-blue)](https://clearpact.polsia.app/docs)
[![Status](https://img.shields.io/badge/Base_Sepolia-Testnet_Live-orange)](https://sepolia.basescan.org/address/0xe1E45CD17FdEe0C38b011FDDA25B259B26140DB0)

**Escrow-native conditional settlement for agent-to-agent payments.**

The glue layer that makes AI-agent payments enforceable: funds held in escrow,
released on verified delivery, settled in stablecoins on Ethereum.

We integrate with [x402](https://www.x402.org), AP2, Kite, AgentPay, and
[ERC-8004](https://eips.ethereum.org/EIPS/eip-8004) — we don't replace them.
We own the piece none of them answers on its own: *how does an agent pay for
work before it's done, and how do funds release after verification?*
